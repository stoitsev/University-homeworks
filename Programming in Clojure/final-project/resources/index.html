Clojure NLP web service

This is web service for natural language processing written in Clojure.

Supported operations

  • Echo (GET http://cljnlp.herokuapp.com/echo/:request) - Outputs the :request string. Ideal for connectivity testing.
  • Tokenize (GET http://cljnlp.herokuapp.com/tokenize/:request) - Contert the the string inside :request into a sequence of tokens.
  • Frequency (GET http://cljnlp.herokuapp.com/frequency/:request) - Return the frequency of each char in the string inside :request.
  • Keywords (GET http://cljnlp.herokuapp.com/keywords/:request) - Return the keywords for the text inside :request.
